v1.1 изменен код под бренды
Для примера из файла brand-test.xlsx подгружаются тестовые данные с id брендами.
(данные по id можно найти на сайте озона меняя фильтры брендов на нужные)

1.установить библиотеки из requirements.txt
2.запустить скрипт ozon-parser.py
3.выполнить дальнейшие указания из консоли
4.после завершения работы скрипта посмотреть новые txt файлы

Для теста не советую парсить много страниц,
так как ожидание окончания парсинга может сильно затянуться.
