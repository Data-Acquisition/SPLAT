Для примера из файла brand-id-test.xlsx(wb brand id) и brand-test.xlsx(ozon brand id)
подгружаются тестовые данные с id брендами.
(данные по id можно найти на соответсвующих сайтах меняя фильтры брендов на нужные)

1.установить библиотеки из requirements.txt
2.настроить локальную бд на сервере
3.запустить скрипты wb_parcer.py, ozon-parser.py
4.выполнить дальнейшие указания из консоли
